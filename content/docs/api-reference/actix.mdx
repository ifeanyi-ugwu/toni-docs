## ActixAdapter

```rust
use toni_actix::ActixAdapter;

ActixAdapter::new() -> ActixAdapter
```

Pass to `app.use_http_adapter` after `ToniFactory::create`:

```rust
let mut app = ToniFactory::create(AppModule).await;
app.use_http_adapter(ActixAdapter::new(), 8080, "0.0.0.0").unwrap();
app.start().await;
```

The API surface is identical to `toni-axum`. Switching adapters requires only changing the import and the adapter constructor call.
