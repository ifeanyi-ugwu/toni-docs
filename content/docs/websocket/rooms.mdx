---
title: Rooms
description: Group WebSocket clients into named rooms and broadcast to specific groups.
---

Rooms let you group clients by a logical name and broadcast to a specific group. Common use cases: chat channels, game lobbies, collaboration sessions, topic subscriptions.

Use `BroadcastService` (from `BroadcastModule`) to manage room membership and send targeted messages.

## Basic room gateway

Declare gateways with the `#[websocket_gateway]` macro. Inject `BroadcastService` for room management:

```rust
use toni::websocket::{BroadcastModule, BroadcastService, WsClient, WsError, WsMessage};
use toni::WsHandlerResult;
use toni_macros::{module, websocket_gateway};

#[websocket_gateway("/rooms", pub struct RoomGateway {
    broadcast: BroadcastService,
})]
impl RoomGateway {
    pub fn new(broadcast: BroadcastService) -> Self {
        Self { broadcast }
    }

    #[on_connect]
    async fn handle_connect(&self, client: &WsClient) -> Result<(), WsError> {
        self.broadcast
            .to_client(&client.id)
            .send(WsMessage::text("Connected. Send join:<room> to join."))
            .await
            .ok();
        Ok(())
    }

    #[subscribe_message("join")]
    async fn handle_join(
        &self,
        client: WsClient,
        message: WsMessage,
    ) -> WsHandlerResult {
        let text = message
            .as_text()
            .ok_or_else(|| WsError::InvalidMessage("Expected text".into()))?;

        let room = text.trim_start_matches("join:");
        self.broadcast.join_room(&client.id, room)?;

        self.broadcast
            .to_client(&client.id)
            .send(WsMessage::text(format!("Joined room: {room}")))
            .await?;

        // Notify others in the room
        self.broadcast
            .to_room(room)
            .send(WsMessage::text(format!("{} joined", client.id)))
            .await?;

        Ok(None)
    }

    #[subscribe_message("leave")]
    async fn handle_leave(
        &self,
        client: WsClient,
        message: WsMessage,
    ) -> WsHandlerResult {
        let text = message
            .as_text()
            .ok_or_else(|| WsError::InvalidMessage("Expected text".into()))?;

        let room = text.trim_start_matches("leave:");
        self.broadcast.leave_room(&client.id, room)?;

        self.broadcast
            .to_client(&client.id)
            .send(WsMessage::text(format!("Left room: {room}")))
            .await?;

        Ok(None)
    }

    #[subscribe_message("message")]
    async fn handle_message(
        &self,
        client: WsClient,
        message: WsMessage,
    ) -> WsHandlerResult {
        let text = message
            .as_text()
            .ok_or_else(|| WsError::InvalidMessage("Expected text".into()))?;

        // Format: "message:<room>:<text>"
        if let Some(rest) = text.strip_prefix("message:") {
            if let Some((room, msg)) = rest.split_once(':') {
                self.broadcast
                    .to_room(room)
                    .send(WsMessage::text(format!("[{room}] {}: {msg}", client.id)))
                    .await?;
            }
        }

        Ok(None)
    }

    #[on_disconnect]
    async fn handle_disconnect(&self, client: &WsClient) {
        // Notify all rooms the client was in
        let rooms = self.broadcast.get_client_rooms(&client.id);
        for room in rooms {
            self.broadcast
                .to_room(&room)
                .send(WsMessage::text(format!("{} left", client.id)))
                .await
                .ok();
        }
    }
}

#[module(
    imports: [BroadcastModule::new()],
    providers: [RoomGateway],
)]
pub struct RoomsModule;
```

## BroadcastService API

```rust
// Room membership
broadcast.join_room(&client_id, room_name) -> Result<(), WsError>
broadcast.leave_room(&client_id, room_name) -> Result<(), WsError>
broadcast.get_client_rooms(&client_id) -> Vec<String>
broadcast.get_room_clients(room_name) -> Vec<String>

// Targeting (returns a sender builder)
broadcast.to_all()                  // all connected clients
broadcast.to_room(room_name)        // all clients in a named room
broadcast.to_client(client_id)      // a specific client

// Sending (on the builder)
.send(WsMessage) -> Result<usize, WsError>          // returns count delivered
.send_event(event, &data) -> Result<usize, WsError> // JSON envelope: {"event":..., "data":...}
```

## Broadcasting from services

`BroadcastService` is `Clone` — inject it into any service that needs to push to rooms:

```rust
#[injectable(pub struct ChatService {
    #[inject]
    broadcast: BroadcastService,
})]
impl ChatService {
    pub fn new(broadcast: BroadcastService) -> Self { Self { broadcast } }

    pub async fn send_to_channel(&self, channel: &str, sender: &str, text: &str) {
        self.broadcast
            .to_room(channel)
            .send(WsMessage::text(format!("[{channel}] {sender}: {text}")))
            .await
            .ok();
    }
}
```

Add it to the same module as the gateway:

```rust
#[module(
    imports: [BroadcastModule::new()],
    providers: [RoomGateway, ChatService],
)]
pub struct ChatModule;
```

## Private messages

Target a specific client by their ID (available from `WsClient.id`):

```rust
#[subscribe_message("dm")]
async fn handle_dm(
    &self,
    client: WsClient,
    message: WsMessage,
) -> WsHandlerResult {
    // parse target ID and text from message...
    self.broadcast
        .to_client(&target_id)
        .send(WsMessage::text(format!("DM from {}: {text}", client.id)))
        .await?;
    Ok(None)
}
```

## Excluding the sender

`BroadcastService` does not have a built-in "broadcast except sender" method. Iterate room members manually:

```rust
let room_clients = self.broadcast.get_room_clients(room);
for peer_id in room_clients {
    if peer_id != client.id {
        self.broadcast
            .to_client(&peer_id)
            .send(msg.clone())
            .await
            .ok();
    }
}
```
